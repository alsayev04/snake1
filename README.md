# snake1 
Я решил создать классическую аркадную игру «Змейка», где игрок управляет змейкой.
1. Основные правила в игре змейки

1.1 Змейка растет при поедании еды(красных квадратов);

1.2 Проигрывает при столкновении со стеной или собственным телом;
1.3 Набирает очки за каждый съеденный кусочек и увеличивается в размерах
3. Управление и визуальные элементы
2.1 Управление — стрелками клавиатуры:
•	← — влево;
•	→ — вправо;
•	↑ — вверх;
•	↓ — вниз.
Змейка не может развернуться на 180° за один шаг (например, сразу из движения вправо перейти к движению влево).
2.2 Визуальные элементы:
•	змейка — зелёные квадраты 20×20 px;
•	еда — красные квадраты 20×20 px;
•	фон — белый;
•	счёт — чёрный текст в левом верхнем углу.
4. Среда программирования: Visual studio code.
5. Язык программирования: Python.
6. Игровой движок: библиотека Pygame.
7. Операционная система: Windows.
