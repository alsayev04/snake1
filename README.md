# snake1 
Я решил создать классическую аркадную игру "Змейка", где игрок управляет змейкой.
1. Основные правила в игре змейки <br>
   1.1 Змейка растет при поедании еды(красных квадратов);<br>
   1.2 Проигрывает при столкновении со стеной или собственным телом;<br>
   1.3 Набирает очки за каждый съеденный кусочек и увеличивается в размерах<br>
4. Среда программирования: Visual studio code.
6. Язык программирования: Python.
7. Игровой движок: библиотека Pygame.
8. Операционная система: Windows.
